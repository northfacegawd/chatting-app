# 간단한 1:1 채팅 사이트

## 개요

로그인한 사용자끼리 1:1 채팅을 할 수 있는 사이트 입니다.

## 개발

### 구현 사항
* 소셜 로그인 구현
* 채팅 생성 구현
* 유저 검색 구현
* 채팅방을 선택 구현
* 1:1 채팅 구현
* 안 읽은 메시지 알림 구현 (실시간으로 바뀔 수 있게 개발중)
